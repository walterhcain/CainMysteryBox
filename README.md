# CainMysteryBox
Unturned plugin made for RoboKast. Permission based boxes used with donations.

## How it works
Using any Unturned donation plugin, you can configure different boxes to give the player. Players who donate, get added to a permission group that has the box. Using the unwrap command will give them a random item out of that box and remove that permission.

Name of box must match the name of the permission group set up in the server. Ex. Box with "Sniper" needs to have the permission group named "Sniper".

## Commands
  ```/boxes``` - Checks which boxes are given to the player<br>
  ```/unwrap <box name>``` - Unwraps the box that is named and gives the player a random item from the box. Removes them from that permission group as well.<br>
